Estrutura básica do HTML

Clique aqui para retornar ao início.

Um arquivo em HTML é composto, resumidamente, por tags. As tags que compoem a estrutura mais elementar de um arquivo em HTML são as tags html, head e body. A tag html delimita o inicio e fim dos códigos que compoem o arquivo HTML. Os elementos que estão dentro da tag head serão carregados antes do usuário visualizar a página. Já os elementos dentro da tag body são os elementos que serão vistos pelo usuário.


Falando sobre tags

Clique aqui para retornar ao início.

As tags são elementos usados para estruturar e formatar o conteúdo da página. Sao escritas entre colchetes <> e podem necessitar serem fechadas. São fundamentais para a construção de uma página.


Atributos das tags

Clique aqui para retornar ao início.

Os atributos são usados para fornecer informações adicionais sobre um elemento específico. Utilizamos atributos dentro das tags para modificar seu comportamento/aparência. Alguns atributos são necessários para o correto funcionamento de algumas tags, como o atributo href na tag a.


Textos

Clique aqui para retornar ao início.

Existem diversos tipos de tags para formatação de textos em HTML, como a formatação de cabeçalhos, parágrafos, subtítulos, títulos etc. Além disso também existem as tags de manipulação de "estilo" textual, como itálico, negrito, sublinhado, etc.


Listas ordenadas e não ordenadas

Clique aqui para retornar ao início.

Em HTML também podemos inserir listas ordenadas e não ordenadas nas páginas, sendo ol e ul respectivamente as tags necessárias para adicioná-las. Para adicionar itens nestas listas devemos usar a tag li. Nas listas podem ser inseridos quaisquer itens desde links, imagens, outras listas, etc.


Links

Clique aqui para retornar ao início.

Os links são bastante úteis para as páginas HTML, pois os mesmos auxiliam na navegação dentro e fora da página, trazendo uma maior interatividade e responsividade da página para com o usuário.
